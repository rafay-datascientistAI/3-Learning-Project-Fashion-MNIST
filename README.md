## 👕 Fashion MNIST Image Classification using CNN

This is a beginner-friendly deep learning project where I implemented a Convolutional Neural Network (CNN) to classify images from the Fashion MNIST dataset.

---

## 📌 About

This project is part of my learning journey in Deep Learning.  
I built a simple CNN model to classify different types of clothing images.

---

## 📊 Dataset

- Fashion MNIST dataset  
- Contains grayscale images of clothing items  
- Classes include shirts, trousers, shoes, bags, etc.  

---
